# 🧀 Dadinhos da Giu — Sistema de Pedidos

Sistema web completo para gestão de pedidos de dadinhos artesanais, desenvolvido para otimizar a operação, controle e crescimento de um pequeno negócio.

---

## 🚀 Sobre o projeto

Este projeto foi criado com o objetivo de digitalizar e automatizar o processo de pedidos da **Dadinhos da Giu**, permitindo:

* Recebimento de pedidos online
* Organização operacional via painel administrativo
* Controle de status de pedidos
* Gestão de estoque
* Visualização em tempo real da operação

👉 O sistema já está em produção e sendo utilizado na prática.

---

## 🌐 Acesso ao sistema

🔗 https://giu-dadinhos-app.vercel.app

---

## 🧠 Funcionalidades

### 🛒 Cliente

* Criação de pedidos de forma simples e rápida
* Seleção de produtos (250g / 500g)
* Preenchimento automático de endereço via CEP
* Escolha de data para entrega
* Envio direto para WhatsApp

---

### ⚙️ Administrativo

* Visualização de todos os pedidos
* Atualização de status:

  * Recebido
  * Pronto
  * Saiu para entrega
  * Entregue
  * Cancelado
* Kanban de pedidos (arrastar e soltar)
* Controle de estoque por produto
* Visualização detalhada de pedidos

---

## 🧩 Tecnologias utilizadas

* **Next.js** (App Router)
* **React**
* **TypeScript**
* **Prisma ORM**
* **PostgreSQL (Neon)**
* **Vercel (Deploy)**
* **Tailwind CSS**

---

## ⚙️ Como rodar o projeto localmente

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/giu-dadinhos-app.git

# Entrar na pasta
cd giu-dadinhos-app

# Instalar dependências
npm install

# Rodar o projeto
npm run dev
```

---

## 🔐 Variáveis de ambiente

Crie um arquivo `.env` na raiz com:


## 💡 Motivação

Este projeto nasceu com o objetivo de resolver um problema real: organizar e escalar o negócio da minha esposa, **Giu**, de forma simples, eficiente e acessível.
